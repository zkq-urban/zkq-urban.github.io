# 学术主页

欢迎访问我的个人学术主页，展示我的研究工作、出版物和项目。

## 概述

这是一个用于展示个人学术成果的网站模板，包含以下部分：

- 个人简介
- 教育经历时间线
- 出版物列表
- 研究项目
- 联系方式

## 技术特性

- 响应式设计，适配各种设备
- 平滑滚动导航
- 交互式元素
- 现代化的UI设计

## 使用说明

要使用此模板，请按照以下步骤操作：

1. 将此项目克隆到您的GitHub账户
2. 修改 `index.html` 中的内容以匹配您的个人信息
3. 替换 `images/profile.jpg` 为您自己的照片
4. 在GitHub仓库设置中启用GitHub Pages

## 自定义内容

您需要更新以下内容：

- 个人简介和研究兴趣
- 教育经历
- 出版物列表
- 研究项目
- 联系信息和社交媒体链接

## 部署到 GitHub Pages

1. 将此项目推送到您的GitHub仓库
2. 在仓库设置中，向下滚动到 "Pages" 部分
3. 选择源分支为 "main" 或 "master"，目录选择 "/ (root)"
4. 点击 "Save" 保存设置
5. GitHub 将自动部署您的网站

您的网站将在 `https://<your-username>.github.io/<repository-name>` 上可用。
